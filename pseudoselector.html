CSS Pseudo selectors:

  • Hover
  • applies when you hover to the element
  • first-child
  • applies to first child of that element
  • last-child
  • applies to last child element
  • nth-child(n)
  • applies to nth child
  • only-child
  • applies to element which has only one child otherwise it won't work
  • You can apply only-child pseudo here since its the only child of ul

There are varieties of pseudo selectors

Please visit the website for more information

You can apply pseudo selectors to id,class or element itself
